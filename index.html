📊 Core Models

User Model

Represents all system users including administrators, vendors, and affiliates

id - String (Primary Key)
email - String (Unique)
password - String
name - String
role - UserRole (ADMIN, VENDOR, AFFILIATE)
referralCode - String (Unique)
totalEarnings - Float
availableBalance - Float
profileImage - String (Optional)
phoneNumber - String (Optional)
bankDetails - String (Optional)
isActive - Boolean (Default: true)
isVerified - Boolean (Default: false)
vendorStatus - VendorStatus (PENDING, APPROVED, REJECTED, SUSPENDED)
createdAt - DateTime
updatedAt - DateTime

Product Model

Represents vendor products available in the platform

id - String (Primary Key)
vendorId - String (Foreign Key)
title - String
description - Text
price - Float
commissionPercentage - Float (Default: 10%)
imageUrl - String (Optional)
category - String (Optional)
isActive - Boolean (Default: true)
createdAt - DateTime
updatedAt - DateTime

Transaction Model

Tracks all purchases and commission calculations

id - String (Primary Key)
buyerId - String (Foreign Key)
productId - String (Foreign Key)
referrerId - String (Optional, Foreign Key)
purchaseAmount - Float
commissionAmount - Float
status - TransactionStatus
transactionType - String (purchase, refund, manual_adjustment)
notes - Text (Optional)
createdAt - DateTime
updatedAt - DateTime

WithdrawalRequest Model

Manages user withdrawal requests and approvals

id - String (Primary Key)
userId - String (Foreign Key)
amount - Float
status - WithdrawalStatus
bankDetails - Text (Optional)
approvedBy - String (Optional)
approvalDate - DateTime (Optional)
rejectionReason - String (Optional)
createdAt - DateTime
updatedAt - DateTime

ReferralClick Model

Tracks referral link clicks for analytics

id - String (Primary Key)
referrerCode - String
productId - String
userId - String (Optional)
ipAddress - String (Optional)
userAgent - String (Optional)
createdAt - DateTime

🏷️ Enumerated Types

UserRole

ADMIN VENDOR AFFILIATE

VendorStatus

PENDING APPROVED REJECTED SUSPENDED

TransactionStatus

PENDING COMPLETED FAILED REFUNDED

WithdrawalStatus

PENDING APPROVED REJECTED COMPLETED
